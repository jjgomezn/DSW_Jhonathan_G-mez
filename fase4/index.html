Propósito del Sitio:

El propósito del sitio es aprender y fomentar el uso de las herramientas de Flex Box y las Listas y enlaces en HTML, utilizando el software Visual Studio Code. Los Flex Box son para: Diseños flexibles y menos código, alinear los elementos y reordenar contenido. Las Listas en HTML nos permite crear conjuntos de elementos en forma de lista dentro de una página, todos los cuales irán precedidos, generalmente, por un guión o número. Los tipos de listas en HTML son los siguientes: Listas ordenadas. Listas desordenadas.
